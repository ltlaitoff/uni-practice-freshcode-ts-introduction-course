# Freshcode TypeScript Introduction Course task

This repository contains a solution for final task of Freshcode TS course

I choose to use `Vite` as a server instead of `tsc --watch` with live-server because it's easier and will work in the same way

Original task description was too detailed and too restrictive, so I changed it to more like Jira ticker from the manager. New description is located in [TASK_DESCRIPTION.md](TASK_DESCRIPTION.md)

IMO it's much easier to understand this one, if we compare it to the original with "create variable, create function" etc

All code was written with a lot of assetions, so we can be sure that our values is what we think they are

To run code you can clone repo, use pnpm
Or, if you don't want to or afraid of any security problems I also have a deployed version on Vercel: https://uni-practice-freshcode-ts-introduct.vercel.app/
