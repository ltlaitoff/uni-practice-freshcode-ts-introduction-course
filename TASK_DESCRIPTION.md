# Original description
ПРАКТИЧНЕ ЗАВДАННЯ

1. Підготуй проєкт: ініціалізуй новий проєкт, встанови компілятор TypeScript
2. HTML-структура:
- Створи файл index.html з `<button>` для запуску скрипта, `<select>` для вибору кольору сторінки та порожнім блоком `<div>` для виведення результату.
3. Робота у файлі TypeScript:

- Оголоси Enum із трьома кольорами.

- Створи кілька змінних різних типів: імʼя - рядок, вік - число.

- Напиши функцію, яка приймає ім'я користувача і вік, та повертає рядок-привітання.

- Оброби подію натискання на кнопку:

- Виведи у створений div привітання користувача.

- Перебери Enum у циклі і виведи всі його значення у вигляді списку.

- Зміни колір фону сторінки на основі обраного значення у селекті.

4. Компіляція та перевірка:

- Скомпілюй TypeScript-код у JavaScript.

- Підключи зкомпільований файл у HTML і перевір роботу скрипта в браузері.

5. * Додаткове завдання


Додай ще два поля вводу:

- одне `<input type="text">` для введення хобі,

- одне `<input type="number">` для введення кількості років досвіду у цьому хобі.


Зміни функцію привітання так, щоб вона враховувала ім’я, вік, хобі та досвід, і повертала розширене повідомлення.

Використай умовну логіку:

- якщо досвід більше 5 років → "Вау, ти справжній експерт у [хобі]!"

- якщо досвід від 1 до 5 років → "Чудово, ти вже маєш досвід у [хобі]."

- якщо 0 або менше → "Все попереду! Починати нове хобі — це цікаво."


Приклад виводу у div:

"Привіт, Аліно! Тобі 22 роки. Твоє хобі — фотографія. Вау, ти справжній експерт у фотографія!"

# New description

Потрібно створити просту веб-сторінку з формою, яка збирає дані користувача і після натискання кнопки відображає персоналізоване привітання та додаткову інформацію.

## Форма містить

Текстове поле — ім'я користувача
Числове поле — вік
Текстове поле — хобі
Числове поле — кількість років досвіду у хобі
Дропдаун (`<select>`) — вибір кольору фону сторінки (3 варіанти)
Кнопка — "Запустити"

## Після кліку на кнопку

Під формою з'являється блок з текстом виду:

"Привіт, Аліно! Тобі 22 роки. Твоє хобі — фотографія. Вау, ти справжній експерт у фотографія!"

Текст про досвід змінюється залежно від введеного числа:

- 5 років → "Вау, ти справжній експерт у [хобі]!"
- 1–5 років → "Чудово, ти вже маєш досвід у [хобі]."
- 0 або менше → "Все попереду! Починати нове хобі — це цікаво."

Нижче виводиться список усіх доступних кольорів з дропдауну (просто перелік значень).
Фон сторінки змінюється на колір, обраний у дропдауні.

## Технічні вимоги

Проєкт написаний на TypeScript, скомпільований у JS і підключений до HTML
Кольори оголошені через Enum
Типізовані змінні та функція привітання
